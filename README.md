# File Expiry Cleaner

A simple project to automatically delete files or folders after a specified period of time **if they are not confirmed**.

This repository provides **two different implementations**, maintained in separate branches, so users can choose the approach that best fits their environment.

---

## Repository Structure

This repository contains **two main branches**:

### 1. Bash Script Version
**Branch:** `bash-script`

- A lightweight solution using a simple **Bash script**
- Suitable for Linux servers or cron-based environments
- No external dependencies required
- Includes a **separate Markdown file** with detailed usage instructions
